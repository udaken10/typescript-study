---
description: 提出された演習コードをレビューします
---

ユーザーが演習問題（exercise.ts）を解いた後、コードをレビューしてフィードバックを提供します。

### ステップ

1. **提出コードの読み込み**
   - ユーザーが取り組んでいるフォルダの `exercise.ts` を読み込みます。

2. **実行と検証**
   - ターミナルで `npm run run:ex curriculum/[フォルダ名]/exercise.ts` を実行します。
   - コンパイルエラーがないか、バリデーションがパスしているかを確認します。

3. **コードの品質レビュー**
   - 以下の観点でチェックします：
     - 型注釈が適切か（`any` が乱用されていないか）
     - 問題の意図通りに実装されているか
     - より簡潔、または安全な書き方がないか

4. **フィードバックの提供**
   - 良かった点、修正が必要な点、学習のアドバイスを分かりやすく伝えます。
   - // turbo
     `npm run save -- "Completed exercise [章番号]"` を実行して、履歴を保存することを提案します。

---

このコマンドを使用するには `/review` と入力してください。
