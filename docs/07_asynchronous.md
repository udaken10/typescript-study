# 07. 非同期処理（待ち時間を上手に使おう）

プログラムの中には、時間がかかる処理（ネットからデータをダウンロードするなど）があります。
その間ずっと止まってしまうのではなく、**「終わったら教えてね！」**と頼んでおいて、他のことをするのが**非同期処理（ひどうきしょり）**です。

## 1. 非同期処理のイメージ
料理に例えると分かりやすいです。
- **同期処理**: お湯が沸くまで、コンロの前でじっと動かずに待つ。
- **非同期処理**: お湯を火にかけている間に、野菜を切る。お湯が沸いたら（通知が来たら）パスタを入れる。

## 2. Promise と async / await
最近は、**async**（アシンク）と **await**（アウェイト）という魔法の言葉をよく使います。

```typescript
// 料理をする関数（3秒かかる）
const cook = async () => {
  console.log("お湯を沸かします...");
  
  // 「終わるまで待っててね」とお願いする
  await wait3Seconds();
  
  console.log("お湯が沸きました！パスタを入れます。");
};
```

## 3. なぜ TypeScript で書くの？
「どんなデータが返ってくるか」が型でわかるので、ネットからのデータ読み込みでよくある「データの名前を間違えて、エラーで止まっちゃった！」というミスを、コードを書いている最中に見つけることができます。
