# 06. ジェネリクス（どんな型でも使える万能テンプレート）

ジェネリクスは一見難しそうですが、**「後から型を決められる仕組み」**だと考えると分かりやすくなります。

## 1. 型の入れ物をつくる
例えば、中身をそのまま返す関数。
`number` 専用、`string` 専用と作るのは大変ですよね。

```typescript
// <T> は「あとで決める型」という意味の仮の名前
function whatIsThis<T>(arg: T): T {
  return arg;
}

// 使うときに型を決める
let num = whatIsThis<number>(123);
let str = whatIsThis<string>("こんにちは");
```

## 2. なぜこれを使うの？
「どんな型でもOK」にするなら `any` でいいじゃない、と思うかもしれません。
でも `any` を使うと、中身が何だかわからなくなってしまいます。

ジェネリクスなら、**「入れるときは何でもいいけど、一度決まったらその型としてしっかり守るよ！」**という安心感があるのです。
